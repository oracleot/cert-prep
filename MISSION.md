# Mission: Learn LangGraph Agent Architecture Through Rex and Sage

## Why
You want to learn LangGraph/LangChain agent architecture by using this repo's Rex/Sage tutoring loop as the concrete example, so you can reason about real agent systems rather than abstract diagrams.

## Success looks like
- You can explain how Rex and Sage are represented as graph nodes, not autonomous black boxes.
- You can trace one user answer through state, evaluation, routing, Sage response, persistence, and the next challenge.
- You can modify or extend a simple LangGraph agent loop without losing track of state and control flow.

## Constraints
- Current LangGraph comfort: basic.
- Lessons should stay short, concrete, and tied to the existing Rex/Sage implementation.
- Prefer this repo's executable code as the primary example.

## Out of scope
- Multi-exam generalization.
- Clerk/auth wiring.
- Advanced LangGraph abstractions before the explicit Rex/Sage state machine is understood.
